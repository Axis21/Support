# Support

Welcome to the public support repository for our platform.

This repository is used for:

- Bug reports
- Feature requests
- Questions
- General product feedback

Our platform helps vibration analysts monitor rotating assets, review vibration data, and improve maintenance decisions.

## Before opening an issue

⚠️ **Public repository:** Anything you submit here can be read by others. Do not include passwords, API keys, access tokens, customer data, or confidential project information.

For confidential or account-specific support, please contact us through your usual support channel instead of posting here.

## Open an issue

Use one of the available issue templates:

- 🐞 **Bug report** — report something that is not working as expected
- 💡 **Feature request** — suggest an improvement for vibration analysis workflows
- 💬 **Other** — ask a question or share general feedback
